# Upload to a Fresh GitHub Repository

1. Extract `EdgeBoard-3.5-Render-Ready.zip` on your computer.
2. Open the extracted folder.
3. Select all files and folders inside it, including `apps`, `docs`, and `render.yaml`.
4. Upload those items to the root of your GitHub repository.
5. Confirm GitHub shows `render.yaml` on the first page of the repository—not inside another folder.
6. In Render, create a new Blueprint from that repository.
7. Use branch `main` and Blueprint Path `render.yaml`.

Do not upload the ZIP itself as the only repository file.
